# Django Pizza

## Setup

The first thing to do is to clone the repository:

```sh
$ git clone https://github.com/chromium7/django_pizza.git
$ cd django_pizza
```

Build the container with docker-compose
```sh
$ docker-compose up --build
```

And navigate to `http://127.0.0.1:8000/`.


## Tools used:
- Django
- PostgreSQL
- Docker
- Git
